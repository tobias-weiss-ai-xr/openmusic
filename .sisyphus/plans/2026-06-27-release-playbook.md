# Release Playbook: First Mix (Jun 27)

> **Goal:** Generate "Deep Minimal Dub" mix and upload to YouTube.
> **Scheduled:** Jun 27, 2026 — @125BPM, Dm, 2h

---

## Prerequisites

1. **GPU machine** with ACE-Step 1.5 cloned + venv set up (see `AGENTS.md#Setup`)
2. **OpenMusic installed** into ACE-Step's venv (`uv pip install -e packages/core`)
3. **FFmpeg** available system-wide
4. **YouTube OAuth** configured: `client_secrets.json` at repo root
5. **TS effects engine** built: `pnpm build` or `npx tsc -p packages/effects/tsconfig.json`

## Command

```bash
ACE-Step-1.5/.venv/bin/python -m openmusic.cli.main release \
  --length 2h \
  --bpm 125 \
  --key Dm \
  --output deep_minimal_dub_2h.flac \
  --title "Deep Minimal Dub Techno | 2 Hours @125BPM" \
  --description "AI-powered deep minimal dub techno mix.\nGenerated by OpenMusic\nhttps://graphwiz.ai\nhttps://github.com/tobias-weiss-ai-xr/openmusic" \
  --tags "dub,techno,minimal,deep,ai,openmusic,ambient" \
  --privacy unlisted \
  --client-secrets client_secrets.json
```

## Pipeline (3 steps, automated)

| Step | What happens | Est. time |
|------|-------------|-----------|
| 1/3 | MixOrchestrator generates 2h mix via ACE-Step DiT | ~40-60min (GPU) |
| 2/3 | FFmpeg loops cover art over audio → MP4 (AAC 256k) | ~2-3min |
| 3/3 | YouTube Upload via OAuth | ~5-10min |

## After Upload

1. **Review** the video at the returned URL (will be `unlisted`)
2. If quality OK → change YouTube privacy to `public`
3. Add video to playlist if desired
4. **Generate shorts** from the mix:
   ```bash
   ACE-Step-1.5/.venv/bin/python -m openmusic.cli.shorts batch-auto \
     --audio mix.flac \
     --mix-length 2h \
     --count 6 \
     --duration 30 \
     --categories stoic,meditation,devops \
     --output-dir shorts/
   ```

## Troubleshooting

| Issue | Fix |
|-------|-----|
| `ffmpeg` not found | `apt install ffmpeg` or `brew install ffmpeg` |
| YouTube OAuth fails | Run `openmusic auth-youtube` first to generate refresh token |
| Mix generation fails | Check CUDA: `nvidia-smi`. ACE-Step needs ~8GB VRAM |
| Cover art missing | `dub_visual.svg` at repo root will be used as fallback |
| `pedalboard` crashes | CPU lacks AVX. Tests mock it; for real generation use TS effects engine |
