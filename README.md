# AI-powered dub techno generation framework

An AI-assisted workflow that combines ACE-Step 1.5, Strudel, and Tone.js to generate dub techno. Includes a CLI for quick generation and export.

CLI usage:

- openmusic generate --length 2h --style dub_techno --output mix.flac

## Installation

- Python 3.12 is required
- Install uv: https://docs.astral.sh/uv/getting-started/installation/
- Install dependencies: `uv sync`
- Install with ACE-Step extras: `uv sync --all-extras`
